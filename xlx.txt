Archivo/Documento	Tipo	Propósito Principal	Documentos Generados	¿Por qué se generan?
preparacion_datos.py	Script Python	Fase 0: Extracción y Limpieza. Descarga datos históricos de clima (NASA POWER), limpia errores y prepara las variables cíclicas (Seno/Coseno) para el entrenamiento.	Reporte_Humano_Huajuapan.csv, Dataset_Entrenamiento_IA.csv	El primero es para validación humana (revisar los datos limpios). El segundo es el input tabular que necesita la LSTM para aprender.
entrenamiento_modelo.py	Script Python	Fase 1: Entrenamiento. Diseña la arquitectura LSTM (con Keras), carga Dataset_Entrenamiento_IA.csv y entrena la red para predecir Temperatura y Lluvia.	mejor_modelo_clima.h5	Es el motor de predicción (el modelo entrenado y guardado). Sin este archivo, no se puede hacer el pronóstico futuro.
mejor_modelo_clima.h5	Modelo Keras/TF	Fase 1: Motor LSTM. Contiene la estructura de la red neuronal y todos los pesos (los conocimientos aprendidos) de la LSTM.	Ninguno (Es la salida de entrenamiento_modelo.py)	Es el activo principal que se usa para generar las predicciones de la Fase 3.
generar_pronostico.py	Script Python	Fase 1.5: Predicción Recursiva. Carga el mejor_modelo_clima.h5 y lo usa para generar una cadena de pronósticos de 900 días. Filtra este pronóstico para dejar solo el año 2026.	Pronostico_2026_IA.csv	Esta tabla de datos es la serie de tiempo de entrada que el Algoritmo Genético (Fase 3) necesita para simular el bienestar del cultivo día a día en 2026.
Dataset_Entrenamiento_IA.csv	Archivo CSV	Input LSTM. Contiene las "ventanas" de 15 días de historia preparadas para entrenar la red (datos de la Fase 0).	Ninguno (Es la salida de preparacion_datos.py)	Es el dataset estructurado en formato secuencial para que la LSTM pueda aprender.
Reporte_Humano_Huajuapan.csv	Archivo CSV	Datos Históricos Limpios. Contiene la historia original de temperatura y lluvia, limpia, y se usa para iniciar la secuencia de predicción en generar_pronostico.py.	Ninguno (Es la salida de preparacion_datos.py)	Es la base de datos de referencia que asegura que el scaler (normalizador) y la fecha de inicio de la predicción sean correctos.
Pronostico_2026_IA.csv	Archivo CSV	Input Algoritmo Genético. Contiene la predicción diaria de temperatura y lluvia para todo el año 2026.	Ninguno (Es la salida de generar_pronostico.py)	Es la tabla que la Función de Aptitud (Fitness) de la Fase 3 usará junto con la Lógica Difusa (Fase 2) para evaluar qué fechas son las mejores para la siembra.
